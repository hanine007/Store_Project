## Setup

### Prerequisites

- Node.js installed
- Database (Mongodb) installed and running

### Installation

1. Clone the repository:

   ```bash
   $ git clone 
   $ cd your-project
   ```

2. Install dependencies:

   ```bash
   $ npm install
   ```
